# Ai_image_generator
image generation app using DALL-E 2.0

Live: [Demo](https://ai-image-generator-amber.vercel.app/)


Deployment: 
  * Backend: render hosting
  * Frontend: Vercel
  
## Frameworks used

### Front-End:

  * React js (vite)
  * Tailwind css
  
### Back-End:

  * Node js
  * MongoDB (For posting)
  * Cloudinary (For Images)
  * Open API
